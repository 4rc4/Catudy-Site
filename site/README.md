# Catudy portfolio site

Bu klasor Catudy icin hazirlanan statik portfolyo/tanitim sitesini icerir.

## Yerel onizleme

```powershell
cd "C:\Users\arca\Documents\New project 3\site"
py -3 -m http.server 5173 --bind 127.0.0.1
```

Ardindan tarayicida:

```text
http://127.0.0.1:5173/
```
